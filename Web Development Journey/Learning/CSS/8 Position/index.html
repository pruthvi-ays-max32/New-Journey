POSITION PROPERTY

Position Property Information
Property's Value Meaning
Static Elements follow the normal document flow. (top, bottom, z-index etc will not work.)
Relative Elements Position adjusted from its normal position.
Absolute Positiona element relative to the nearest postioned ancestor. Comes above the page.
Fixed Elements Position relative to the viewport. Doesn't move or scroll with page.
z-index Determines the stacking order of elements along the z-axis. +ve value : appear on top, -ve value : appear below of other.
