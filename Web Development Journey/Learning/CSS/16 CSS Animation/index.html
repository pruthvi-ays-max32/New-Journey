CSS Animation

CSS Allow animation of html element without using JS.
Animation Properties
Property Meaning
animation-name/td> here we assign name of keyframe we created for animation.
animation-duration total time in which animation should complete one cycle.
animation-timing-function : speed control of animation.
  • ease : start slow then fast and end slow
  • ease-in : slow start
  • ease-out : slow end
  • ease-in-out : slow start and slow end
  • linear : same speed
  • cubic-bezier(n,n,n,n) : let u define your own values.
animation-delay After how many seconds animation should start.
animation-linear-count how many times the animation should takes place.
animation-direction In which direction the animation should takes place.
animation : we can define all 6 prop at once. [transition : animation-name animation-duration animation-timing-function animation-delay animation-linear-count animation-direction]
