Colors Properties In CSS.

There are mainly two color properties in css.

  1. Color : to change the color of text.
  2. Background Color : to change the background of text / content

There are mainly 4 units for declaring the color property.

  • Hashcode : #ABC123
  • hsl (hugh saturation Lightness.) : 120 30% 20%
  • rgb (red green blue) : 0 0 225 (All 3 values must be between 0 - 255)
  • rgba (red green blue alpha) : 0 0 225 0.7 (alpha : Transperancy of color -> between 0 to 1)
