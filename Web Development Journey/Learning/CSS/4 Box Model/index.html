BOX MODEL IN CSS

Box Model Heirarchy : Margin > Border > Padding > Content

Margin : Used to seperate 2 Elements.

Padding : Used to seperate Content from Border.

    padding : 40px ; (all 4 sides)
    padding : 40px 30px; (Top-Bottom && Left-Right)
    padding : 40px 30px 20px; (Top && Left-Right && Bottom)
    padding : 40px 30px 20px 10px; (Top Right Bottom Left)
