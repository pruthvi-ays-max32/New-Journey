Types of pseudo Class

Property value
hover Works When we move cursor on element
link Select all universal link.
Active Works When we move cursor on element and click the button.
Visited Works When we clicked that button in browser previously at least once. (In ignozite mode it will not work because it doesn't store the history.)
nth-child "x:nth-child(n)"Select every x elements that is the nth child of its parent it supports "an+b" -> where a and b are variables.(just change the n value here from 0 to infinite it will select those elements) eg. suppose we have to select all odd number rows from list then "nth-child(2n+1)" etc.
first-child "x:first-child" -> Select every x elements that is the first child of its parent
last-child "x:last-child" -> Select every x elements that is the last child of its parent

Pseudo Elements are used to style specified parts of element.

Eg. first letter of first Word in paragraph. we use "::" for this.
Types of Pseudo Elements
Property value Example
Selection If we try to select any text from our page it will show these properties which we define here. eg. selected text will show background color, color which we will define here. ::selection{background-color : red} -> it will show bg-color red for selected text on page.
first-letter To Style first letter of any word. h1::first-letter{font-size : 30px} -> it will change the font-size of first letter of every h1 tag on page
first-line To style first line of any paragraph. p::first-line{} -> it will style the first line of p tag.
Before If we wanted to display something just before any element. then we use before. but this will not be part of our DOM it will only display but can't be select. It has special property content:"" where we define what we want to display. h3::before{content:"required"} -> it will display required before every h3 tag.
After If we wanted to display something just after any element. then we use after. but this will not be part of our DOM it will only display but can't be select. It has special property content:"" where we define what we want to display. h3::after{content:"required"} -> it will display required after every h3 tag.
