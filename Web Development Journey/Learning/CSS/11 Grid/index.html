Grid Properties

[Multi D layout for arranging item.]

Grid Container Properties
Grid Property Details Example
gap(row-gap, column-gap) it defines the gap between grid items. row-gap:20px;
grid-template-columns define width of each column 
grid-template-columns:1fr 1fr 2fr ...  ;  
            (no of values = no of columns. [fr=fraction])
            grid-template-columns:repeat(4, 1fr);[repeat 4 times 1fr]
grid-template-rows define height of each row grid-template-rows : 1fr 2fr 200px; remaining rows will take default height.(default height can be given by [grid-auto-rows:150px;])
justify-content align content of grid on x-axis
align-content align content of grid on y-axis
align-items align content of grid-item on y-axis inside div of that grid-item.
Grid Child/Item Properties
Grid Property Details Example
grid-column in how many columns that child should display. grid-column-start:1 ; grid-column-end:3; OR grid-column:1/3;
grid-row in how many rows that child should display. grid-row-start:1 ; grid-row-end:3; OR grid-row:1/3; OR grid-row:1/span 3 (span 3 : 3 spans it will occupie)
grid-area in how many rows and columns that child should display. grid-area : row-start / col-start / row-end / col-end
grid-template-area We can give names to each item. 
grid-template-areas : "header header header"
                    "sidebar main main"
                    "sidebar main main"
                        "footer footer footer";

                        #box1{
                          grid-areas:header; (box1 will occupie whole place of
                                           header in grid template area.)
                        }
justify-self to move perticular item's content on x-axis
align-self to move perticular item's content on y-axis
