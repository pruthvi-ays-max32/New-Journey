UNITS IN CSS

Absolute : Doesn't Depend upon anything.

Relative : Dependent upon something or someone.

Information of Units In CSS
Units Type Information
% Relative Related to parent element.
em Relative Related to font size of parent element.
rem Relative Related to font size of root element (2rem = 2 * px of root element).
vh / vw Relative Related to viewport.
px Absolute Mainly Used for defining border.
