CSS Transform

To apply 2D or 3D Transformation to the element.

transform-origin : top; -> it will define from where we have to start transforming.

BOX
Property What will Happen Other Values/Types
rotate(25deg) it will rotate by 25 degree. rotateX , rotateY, rotateZ
Scale(2) it will increase size by 2 times of original. scaleX, scaleY
skew(45deg) it will bend by 45 degree. skewX, skewY
translate(200px) it will shift to right by 200px from original place. (ON X-axis Default) translateY(2px) -> 2px on Y-axis, translate(20px 30px) -> 20px on x-axis and 30px on y-axis.
