CSS Transition

css transition allow you to change property values smoothly, over a given duration.
Property Meaning
transition-property write down which properties we have to do transition. (all, perticular prop also.)
transition-duration : total time in which transition should take place
transition-timing-function : speed control of transition.
  • ease : start slow then fast and end slow
  • ease-in : slow start
  • ease-out : slow end
  • ease-in-out : slow start and slow end
  • linear : same speed
  • cubic-bezier(n,n,n,n) : let u define your own values.
transition-delay After how many seconds transition should start.
transition : we can define all 4 prop at once. transition : transition-property transition-duration transition-timing-function transition-delay
